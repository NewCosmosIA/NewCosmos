Política de Privacidade

Última atualização: Abril de 2026

O NewCosmos foi construído sobre um princípio fundamental: o que você compartilha em um encontro é sagrado e confidencial. Nenhuma pessoa — nem mesmo a equipe NewCosmos — tem acesso ao conteúdo das suas conversas. Esta política explica, de forma clara e honesta, como tratamos seus dados.

1. Quem somos

O NewCosmos é uma plataforma de bem-estar integrativo com inteligência artificial, operada por João Prata, com sede em Vila Velha, Espírito Santo, Brasil. Site: newcosmos.com.br · E-mail: contato@newcosmos.com.br

2. O que acontece com suas conversas

Suas conversas não são armazenadas por nós. O histórico de cada encontro existe apenas na memória do seu navegador durante a sessão e é apagado automaticamente quando você fecha a aba ou encerra o encontro.

As mensagens são processadas em tempo real pela API da Anthropic (criadora do Claude), que possui uma política clara de não usar conversas de clientes via API para treinar seus modelos de inteligência artificial. Isso significa que o que você diz não alimenta sistemas de IA.

Em resumo: Nenhum humano lê suas conversas. Nenhum sistema armazena suas conversas. O que acontece no encontro, fica no encontro.

3. O resumo pós-sessão

Ao final de cada encontro, nosso sistema gera automaticamente um resumo holístico da sessão — contendo insights, práticas recomendadas e orientações para os próximos dias. Este resumo é:

  • Gerado por IA a partir dos temas abordados, sem transcrição literal da conversa
  • Enviado exclusivamente para o e-mail cadastrado no momento da compra
  • Não acessado por nenhuma pessoa da equipe NewCosmos
  • Armazenado de forma segura no banco de dados apenas como metadado da sessão

4. Dados que coletamos

Coletamos apenas o mínimo necessário para o funcionamento do serviço:

  • Nome e e-mail — fornecidos no momento da compra via Hotmart, para envio do link de acesso e resumos pós-sessão
  • Plano e token de acesso — para controlar o número de sessões disponíveis
  • Metadados da sessão — data, duração em minutos e plano utilizado (sem conteúdo)
  • Idioma preferido — salvo no seu navegador via localStorage para personalizar a experiência

Não coletamos dados de localização, dados de navegação ou qualquer informação além do necessário para a prestação do serviço.

5. Compartilhamento de dados

Seus dados são compartilhados apenas com os seguintes serviços, necessários para o funcionamento da plataforma:

  • Anthropic — processamento das mensagens via API (sem armazenamento para treino)
  • Hotmart — plataforma de pagamento e gestão de assinaturas
  • Resend — envio de e-mails transacionais (acesso e resumos)
  • Supabase — banco de dados seguro para tokens e metadados de sessão

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de marketing ou publicidade.

6. Segurança

Todas as comunicações entre seu navegador e nossos servidores são criptografadas via HTTPS. Os tokens de acesso são únicos, gerados aleatoriamente e têm prazo de validade. O banco de dados é protegido por Row Level Security (RLS) do Supabase.

7. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar os dados que temos sobre você
  • Solicitar a correção de dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados da nossa base
  • Revogar o consentimento a qualquer momento

Para exercer qualquer desses direitos, entre em contato pelo e-mail contato@newcosmos.com.br.

8. Retenção de dados

Os metadados de sessão (nome, e-mail, plano, duração) são mantidos pelo período necessário para a gestão das assinaturas e cumprimento de obrigações legais. Após o cancelamento de um plano, os dados são excluídos em até 90 dias.

9. Cookies e armazenamento local

Utilizamos apenas localStorage do navegador para salvar sua preferência de idioma. Não utilizamos cookies de rastreamento, pixels de publicidade ou ferramentas de analytics que identifiquem você individualmente.

10. Menores de idade

O NewCosmos é destinado a pessoas com 18 anos ou mais. Não coletamos intencionalmente dados de menores de idade. Se identificarmos que um menor utilizou o serviço, os dados serão excluídos imediatamente.

11. Alterações nesta política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por e-mail aos usuários ativos. A data da última atualização está sempre indicada no topo deste documento.
